What's new

Two things no one else is doing.

1

Dynamic recruitment

When the Critic flags a compliance risk, the Arbiter pulls a Compliance specialist into the live Band room mid-debate. Agents recruiting agents — not pre-wired.

2

Cross-model courtroom

Debaters run across DeepSeek-V3.1-Terminus, DeepSeek-V3.2 and GPT-4.1-mini over AI/ML API and Featherless — a genuine multi-model panel, not one model talking to itself.
