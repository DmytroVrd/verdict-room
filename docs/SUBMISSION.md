**Short Description (≤255 chars):**

```
Verdict Room turns weeks of vendor due diligence into a 3-minute multi-agent
debate inside a Band room. Six specialized agents research, argue, recruit a
compliance specialist, and issue a scored, defensible BUY / AVOID verdict.
```

**Long Description (≥800 chars — розгорнутий pitch):**
```
Choosing a B2B software vendor means one person juggling pricing, security,
GDPR, migration risk, and alternatives — 20 to 40 hours of work spread over
weeks, ending in a single biased summary with no real review.

Verdict Room replaces that with an AI courtroom that lives inside a Band room.
A user drops a purchase case, and six specialized agents collaborate entirely
through Band: a Researcher gathers sourced facts, a Scout finds alternatives,
an Advocate argues for the purchase, a Critic attacks it, and an Arbiter runs
two structured debate rounds before issuing a scored verdict — BUY,
BUY_WITH_CONDITIONS, or AVOID — with explicit conditions and a dissent.

The standout moment is dynamic recruitment: when the Critic flags a data-privacy
risk, the Arbiter pulls a Compliance specialist INTO the live debate through
Band's participant tools — agents recruiting agents mid-workflow. Debaters run
on different providers (AI/ML API and Featherless), making it a genuine
cross-model panel rather than one model talking to itself.

Band is the actual coordination layer: every handoff, shared fact, debate turn,
recruitment, and verdict happens in the room. This directly addresses the
procurement workflow the hackathon highlights as a strong use case — turning a
slow, subjective process into a fast, transparent, defensible decision.
```
